HubBox - Customer App

An app meant for HubBox customers to have everything they need at only one click away to never miss a delivery again. People are more likely to use an App in order to check his personal details rather than to log in into a web App. It comes with great functionalities. I was a solo developer for the App and some important features are:

  1. Apple maps to display Collect Point locations with a small “raining animation” for the pins
  2. Replicating the Apple maps TableView by dragging it up or down, with animation if you stop dragging it at the middle distance between the closest point (top, middle or bottom of the screen)
  3. TextField validation using icons and error messages
  4. TextField displaying format on focus change
  5. Loading screens when a request to the server is made using “NVActivityIndicatorView” library
  6. Braintree for adding card details
  7. Directions between your current location and Collect Point location
  8. Push notification system (Firebase) and SNS
  9. New Relic integration
  10. Collection View for Retailers page
  11. Code migration from swift 2.2 to swift 3.0 and bug fixing
  12. Alamofire, ObjectMapper and AlamofireObjectMapper for requests.
  13. Account activation using a Magic link sent in an email
